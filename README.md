# Building this website

To run this project locally using Vite, follow these steps:

Clone this repo.
```shell
git clone https://github.com/Team-Thermocline/thermocline.github.io.git
cd thermocline.github.io
```

Install deps
```shell
npm install
```

Run the dev server
```shell
npm run dev
```
