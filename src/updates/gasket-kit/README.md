# Gasket Kit

This update contains all the gaskets as their final versions! We tested lots of different materials, laser cutting settings, pressure, thickness and more. The gaskets may not be perfect or last forever but we hope this collection of design files can aid or inspire!


## Note on silicone cutting settings

From Jacob:

```
S (20), P (100), F (100) --> 100% Cut through
But for multiple passes and a slightly cleaner cut:
S (50), P (50), F (50) --> 70% Cut through then S (50), P (75), F (75) --> 100% Cut through
```
